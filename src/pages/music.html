頁面說明

本頁以音樂播放器 Web Component 為展示核心。初次進入會出現「請點擊任意處」遮罩，並以靜音開始播放，再引導是否開啟音量，降低進站即大聲播放的風險。支援循環、隨機與順序播放，也可從清單選曲；清單與播放邏輯由元件內處理，頁面只需提供資料來源。

重點摘要

核心能力 整合歌單管理、播放控制（播放/暫停、切換、模式切換）與進度/音量調節。具備高度封裝性與樣式隔離，無需依賴第三方框架即可獨立運行。
高度客製 支援全方位樣式自定義，包括圖示選用、清單配色及 UI 佈局調整。可輕易適應不同品牌視覺與設計需求。
操作彈性 提供開發者高度彈性的設定，可預設初始音量、播放模式（循環、隨機）等行為，滿足多樣化的應用情境。
資源授權 示範用音樂與封面來源為公有領域素材，符合作品展示與學習用途。
