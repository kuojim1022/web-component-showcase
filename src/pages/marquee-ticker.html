頁面說明

本頁以 「跑馬燈 Web Component」 為核心展示目標：將多則公告以橫向輪播方式呈現，讓重要資訊在不佔版面的情況下持續曝光。適合用於全站公告、活動訊息與系統維護提醒等情境。

重點摘要

核心能力 整合資料載入與連續滾動顯示，提供可點擊、不中斷的公告跑馬燈體驗。
操作彈性 可彈性調整滾動速度、間距、方向與上下留白等全方位自定義選項，滿足不同使用情境需求。
響應設計 在視窗大小或容器寬度變化時，會自動重新計算需要的內容長度與動畫時間，維持連續且自然的滾動節奏。
封裝相容 採用 Web Components 標準與 Shadow DOM 封裝樣式與結構，不依賴第三方套件，支援現代主流瀏覽器。
